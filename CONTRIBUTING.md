# Contributing

This repo is for public class-safe defensive security learning.

## Contribution Rules

- Keep changes respectful and educational.
- Do not add private student identity mapping.
- Do not add secrets, tokens, credentials, or private evidence.
- Do not add exploit instructions.
- Keep pull requests small and easy to review.
- Link to public class repo targets through the catalog instead of duplicating private notes.

## Good Contributions

- clearer review templates
- safer wording for findings
- small exercise improvements
- links to official framework references
- typo fixes
- examples that are fully public-safe

## Before Opening A Pull Request

Check:

- [ ] No private information is included.
- [ ] No secret values are included.
- [ ] Links are public and intentional.
- [ ] The change fits the class defensive review scope.
- [ ] The wording is professional enough for a shared classroom repo.
