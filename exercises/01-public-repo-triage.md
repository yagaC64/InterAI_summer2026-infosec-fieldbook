# Exercise 01: Public Repo Triage

Goal: learn how to inspect a public class repo without turning the review into noise.

## Task

Pick one repo from the Notes column in the class catalog:

- https://github.com/yagaC64/InterAI_summer2026/blob/main/CLASS_GITHUB_CATALOG.md

Complete [templates/repo-review.md](../templates/repo-review.md) for that repo.

## Questions

1. What does the repo appear to do?
2. Can a new user run or understand it from the README?
3. What files are most important to review?
4. Is there any content that should not be public?
5. What is the smallest useful improvement?

## Output

Create one of:

- a public issue with a safe, respectful suggestion
- a small pull request
- a private note to the instructor if the finding may expose sensitive data
